# LITEBRICK - Лендинг компании по внедрению ИИ

Одностраничный сайт-визитка компании, которая внедряет искусственный интеллект в бизнес-процессы.

## Технологии

- **React 18** - UI библиотека
- **Vite** - Сборщик и dev-сервер
- **CSS3** - Стилизация с использованием CSS-переменных

## Установка и запуск

### Требования

- Node.js 18+ 
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск dev-сервера

```bash
npm run dev
```

Сайт будет доступен по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

### Превью продакшен-сборки

```bash
npm run preview
```

## Дизайн-система

### Цветовая палитра

- **Fuchsia**: `#D946EF` - Основной акцентный цвет
- **Electric Purple**: `#A855F7` - Вторичный акцент
- **Deep Violet**: `#6D28D9` - Глубокий фиолетовый
- **Hot Pink**: `#EC4899` - Яркий розовый
- **Black Purple**: `#1E1B4B` - Тёмный фон

### Типографика

- **Заголовки**: `Space Grotesk` - Жирный, с лёгким letter-spacing
- **Подзаголовки**: `Space Grotesk` - Средний вес
- **Основной текст**: `Inter` - Регулярный вес
- **Акценты (CTA, цифры)**: `Orbitron` - Верхний регистр, с letter-spacing

### Градиент фона

```css
background: linear-gradient(135deg, #D946EF, #6D28D9);
```

## Документы

В папке `public/docs/` должны быть размещены следующие PDF документы:

- `privacy-policy.pdf` - Политика конфиденциальности
- `offer.pdf` - Договор оферты
- `personal-data-consent.pdf` - Согласие на обработку персональных данных
- `marketing-consent.pdf` - Согласие на маркетинговые рассылки

## Интеграция с backend

В текущей версии формы отправляют данные в `console.log`. Для интеграции с backend:

1. В `AuditForm.jsx` - замените `console.log` на вызов API
2. В `Booking.jsx` - замените `console.log` на вызов API

Пример:

```javascript
const response = await fetch('/api/audit', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(formData)
});
```
